QUICK START

  1. In OBS, click + in Sources and select Browser
  2. Name your source (e.g., "Break Timer")
  3. Enter the overlay URL:
    https://gscrl-obs.pages.dev/overlays/break-timer.html
  4. Set dimensions (e.g., 400 x 450 for Break Timer)
  5. Add custom CSS for transparency:
    body { background: transparent !important; }
  6. Position and resize in your scene

URL PARAMETERS

Configure overlays via URL parameters for preset configurations:

Parameter Description Example
duration Timer duration in minutes ?duration=15
label Display text ?label=BACK%20SOON
color Accent color (hex without #) ?color=4ECDC4
autostart Start timer on load ?autostart=true
hidecontrols Hide control buttons ?hidecontrols=true

Example full URL:

https://gscrl-obs.pages.dev/overlays/break-timer.html?duration=10&label=BRB&color=FF6B6B&hidecontrols=true
